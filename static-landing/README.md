# Static Landing Page

This directory is separate from the main Life OS app.

Files:

- `index.html`: standalone static landing page for Hulu

Hosting:

1. Serve the contents of `static-landing/` from any static web server.
2. No build step is required.
3. The page is self-contained and uses inline CSS only.

Note:

- The CTA currently uses placeholder `mailto:` links with `adonay@example.com`.
- Replace those links in `index.html` with your real email address or your actual contact/waitlist URL.
